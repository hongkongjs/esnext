Destructuring
The destructuring assignment syntax is a JavaScript expression that makes it possible to extract data from arrays or objects into distinct variables
Example

Before ES6


//Array
var arr = [10,20];

var arg1 = arr[0];

var arg2 = arr[1];

//Object
var obj = {name:"Superman", job:"Pimp"};

var name = obj.name

var job = obj.job

ES6


      //Array destructuring
      var arr = [10, 20];

      [ arg1, arg2 ] = arr;
      console.log(arg1, arg2);
      //arg1=10, arg2=20


      //Object destructuring
      const obj = { name:"wolverine",age:40, job:"bikini model" };

      ( {name, job} = obj );

      console.log(name,job);

      //name=Superman, job=bikini model
Swapping variables problem
Swaping Variables

Before ES6


            var a=10;
            var b=20;
            var tmp=a;
            a=b;
            b=tmp;

ES6


              var a=10;
              var b=20;
              [a,b]=[b,a];
              console.log(a,b); //20 10
