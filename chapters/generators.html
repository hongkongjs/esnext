What are generators?
Generators are functions which can be exited and later re-entered. Their context (variable bindings) will be saved across re-entrances.

Normal Function


              function printArr(arr){
                for(var i in arr){
                  console.log(arr[i]);
                }
              }
              printArr([1,2,3]);
              //1
              //2
              //3

Generator Function


              /*Notice the asterisk*/
function *printArr(arr){
                    for(let i of arr){
                      yield i;
                    }
                }/*Return Generator Object*/
              const gen=printArr([1,2,3]);console.log(gen.next());
/*{ value: 1, done: false }*/console.log(gen.next());
/*{ value: 2, done: false }*/console.log(gen.next());
/*{ value: 3, done: false }*/console.log(gen.next());
/*
              {value: undefined, done: true }*/
